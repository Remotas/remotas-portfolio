# Portfolio — Melquiades Farías

Portfolio técnico construido con **Next.js 15 (App Router)**, **TypeScript** y **Tailwind CSS**. Contiene páginas de inicio, proyectos, experiencia, certificaciones y contacto (integrado con Resend).

## Stack

- Next.js 15 (app/)
- TypeScript
- Tailwind CSS
- MD / YAML para contenido
- Despliegue en Vercel

## Desarrollo

```bash
pnpm install
pnpm dev
# http://localhost:3000
```
